What I’m good at?

I specialize in crafting seamless, responsive designs that elevate the user experience. My expertise includes:

HTML5 & CSS3

HTML5 is the latest version of HTML, enabling richer web structures and direct multimedia support. CSS3 is the latest version of CSS, allowing for more dynamic web designs with features like animations and responsiveness.

Tailwindcss & Bootstrap

Tailwind CSS offers granular control with utility-first classes, while Bootstrap provides pre-designed components and a responsive grid system. Both streamline web development but cater to different styling approaches.

NextJS

Next.js is a React framework that enables server-side rendering, static site generation, and routing. It’s designed for building fast, optimized, and SEO-friendly web applications with ease, offering features like API routes, automatic code-splitting, and built-in CSS support.

UI/UX Design

UI/UX Design focuses on creating intuitive and visually appealing interfaces (UI) and enhancing the overall user experience (UX). UI design involves the look and feel of the product, while UX design focuses on how users interact with it, aiming to improve usability and satisfaction.
